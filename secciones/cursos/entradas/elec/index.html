mkdev

$ base64 imagen.png 1> archivo.txt

$ cat archivo.txt

32hhadu7asdaghsfsgd7fsdfisadfu82238sdfauidyf98safdhi

$ base64 -d archivo.txt

Explicación

Las líneas de código en la sección de mkdev convierten una imagen en una secuencia de cadenas de caracteres imprimibles. Posteriormente, estas cadenas se transforman de nuevo en la imagen original. Este método es útil para convertir cualquier archivo a formato de texto, permitiendo su envío a través de servicios de mensajería. Una vez recibido en el otro extremo, se reconstruye para recuperar su formato original. Esta técnica puede aplicarse a videos, música, y más; representando una forma diferente de enviar archivos.

Dato curioso

El concepto de "pipes" o tuberías en la línea de comandos permite que la salida de un comando se utilice como entrada para otro, facilitando la combinación de herramientas y la creación de flujos de trabajo complejos con comandos simples. Por ejemplo, se puede tomar la salida de un comando que lista archivos y utilizarla como entrada de otro comando que filtrara la salida.
